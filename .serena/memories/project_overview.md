# Project Overview

目的: エロゲーレビューの API サーバと Web フロント（console/showcase）を提供するプロジェクト。

概要:
- API サーバは FastAPI + SQLModel で console/showcase の2系統を提供。
- Web は Next.js App Router を使用した console/showcase のフロント。
