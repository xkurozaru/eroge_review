# Game Review 管理ページ

## 要件

サイドバーメニューあり
1 Game Spec に対して 0 ~ 1 Game Review
Review が見つからない場合も一覧に表示
ステータスは行の色で表現
タイトルをクリックすると Review がないなら新規作成ページ、あるなら編集ページに遷移

## デザイン

ページタイトル

タイトル（部分一致）検索フォーム ブランド名（部分一致）検索フォーム ステータス（未評価/未公開/公開済） 検索ボタン
Game Review 一覧
|タイトル|ブランド|評価値/期待値(達成率)|レビュー作成日時|レビュー更新日時|
ページャー
